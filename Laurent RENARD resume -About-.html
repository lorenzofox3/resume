  4. Amadeus IT

    January 2012-April 2014,Nice, France

    Web/Software developer

    Amadeus IT is the biggest IT provider for travel industry in Europe and one of the biggest in the world. My mission there was to build a customisation framework. Amadeus IT develop standard web application so that airlines, travel agencies, etc. can connect to their central system to perform booking, check availability and other business operations

    Some of the customers want to customize the standard application. My team was in charge of developing a framework so that these customers could write code (components) to be injected in the standard web applications. These components were versioned and hosted into git repositories by our framework and served as a resource when activated so that the end user could enjoy the feature seamlessly compared to the traditional web applications. We also offered an online development environment for the customizers.

    • The core of the framework was developed in Java
    • The online environment web application was built in Java for the server side and in AriaTemplates (in house MVC javascript Framework) for the client side
    • My team was following the SCRUM methodology with 3 weeks release cycles
